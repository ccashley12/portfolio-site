PING

A React Native app with real-time messaging

PING provides users with a seamless chat experience where they can send messages, share images, and their location. The app uses Google Firebase for real-time data storage and authentication.

Project Objective

The goal of PING is to provide a seamless chat experience with features like:

  • Anonymous login via Firebase Authentication
  • Support for sending images and location sharing
  • Offline message storage using AsyncStorage
  • Real-time messaging with Firestore

Technology Behind PING

  • Expo - Simplified development and testing
  • AsyncStorage - Local storage for offline functionality
  • Gifted Chat - Library for chat UI components
  • Firebase Firestore - Real-time cloud database
  • Firebase Authentication - Secure user login

Challenges [&] Growth

PING was my first mobile app development project. I really enjoyed learning about React Native. There were many skills learned such as creating a firebase storage for the app to cache old messages, designing the app within the JavaScript code, using mobile simulators to see the app throughout development, and lastly using Expo for testing during development. Along with these new skills there were also new challenges to overcome like not being able to use both IOS and Android simulators at the same time so I got creative with how to use both to interact with the app. This tought me technical problem solving skills I was not even aware of before.
