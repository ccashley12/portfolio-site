Flix

An Angular based movie app

Flix is an Angular-based movie app that provides users with details on various movies, directors, and genres. Users can register, log in, and create a list of their favorite movies. Built with Angular Material and connected to an existing RESTful API.

Project Objective

The goal of Flix is to provide users with an intuitive movie browsing platform that allows them to:

  • Register and log in securely
  • Browse a collection of movies
  • View detailed information about movies, directors, and genres
  • Manage a list of favorite movies
  • Manage their account and edit profile information

Technology Behind Flix

  • Angular - Framework for dynamic web applications
  • Angular Material - UI components for a responsive and accessible design
  • RESTful API - Connects to a backend for real-time movie data
  • JWT Authentication - Secure user login and authentication
  • TypeScript - For code maintainability
  • GitHub Pages - For deployment and hosting

Challenges [&] Growth

Developing Flix was a very big challenge for me. This project helped me gain a deeper understanding of the features in Angular, boosting my ability to develop dynamic web applications. I also explored TypeScript to enhance code maintainability and used TypeDoc to create detailed documentation. As a beginner with Angular, I found the official documentation difficult to navigate, which made the learning process more challenging than anticipated. Nonetheless, overcoming these obstacles improved my adaptability and problem-solving skills, reinforcing my knowledge of these technologies.
